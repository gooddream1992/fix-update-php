== Description ==

Earn money by streaming local deals from Groupon, LivingSocial, BuyWithMe, DealOn and more from an easy to install widget.

The Dealn.It DealFeed widget gives you access to deals from Groupon, LivingSocial, BuyWithMe, and many 
more. Visitors to your site will see local deals near them which increases relevancy and conversion.

The widget is customizable:

* Categories
* Display Title
* Number of deals to display
* Image size
* Link target
* Target deals to Lat/Lng
* Dealn.It affiliate ID

We share 100% of the commissions with our affiliates. 
Sign up at [http://dealn.it](http://dealn.it)

== Installation ==

Installation is easy.

1. Sign up as an affiliate at [http://dealn.it](http://dealn.it)
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Or, Install from the Wordpress Plugins Installer
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Wordpress Widgets section and drag the DealFeed widget into your dynamic sidebar
1. Select categories, title, image size, number of deals to display, and most importantly your affiliate ID
1. Save Settings, and start earning
